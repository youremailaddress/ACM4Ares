## 小C的奇幻之旅

### 第一章
- 这是小C第一次独自来到离家这么远的城市。S市坐落在海滨，距铁路那头上千公里。凌晨五点多，小C就拖着大包小包的行李走出了车站。这是他大学的第一天。他曾经设想的一家人帮忙送上学的场景也因为今年的疫情泡汤了，走在空无一人的街道上，他摘下来口罩，想着，虽然冷清了点，但看起来似乎也还不错。小C考上的大学离海非常近，据说站在学校最高的楼上，可以看到远方港口正在卸下的集装箱。仲夏的清晨也还没有那么热，越往海边走越能感觉到从陆地吹向海洋的离岸风。看着海平面上刚跃出地平线的太阳，小C感觉到说不出的高兴。似乎又回到了几个月之前高考出分之后，他以H省某市重点学校第三名的好成绩被全市报导的那种不真切感之中。自己居然真的如愿以偿的来到了自己心心念念好久的学校。他不知道的是，他即将卷入一场奇幻之旅。

- 远远的出现了学校的大门，小C加快了脚步，背上的书包背的他肩膀有点酸痛。眼看就要到学校门口了，一位看上去挺顺眼的小姑娘挡在了他面前。小C吞了口口水，和女生说话可能是他这辈子最不擅长的事情了。她要干嘛？问路？我也是刚来的啊？帮忙掂东西的志愿者？啊那太好了...正乱想着，女孩开口了:
- “同学，办卡吗？”

- 盛情难却，盛情难却，小C嘀咕着。然而事实是小C一时间不知道要说什么好，犹豫间就被领到了一边的营业厅。哦豁，这么多种套餐啊？选哪种好呢？
我们这边有三类主流套餐，第一种类是按固定资费，80元/月不限流量和时长。第二类是10元月租，电话费每分钟0.8元，不足一分钟按一分钟计算，流量每G8元，不足1G按1G计算。第三类是...

>小C一个月要打m分钟电话，用nG流量，请你帮忙写一个程序，输入两个浮点数类型数据，判断第一类划算还是第二类划算。若第一类划算，输出A is cheaper than B by xx yuan，若第二类划算，输出B is cheaper than A by xx yuan。若两类一样划算，输出EQUAL。要求不使用除了stdio.h之外的其他库。


```
示例输入:
49.5 2.7
B is cheaper than A by 8.0 yuan.

71.9 2.8
A is cheaper than B by 11.6 yuan.
```


- 嗯，不错不错。小C想着继续往下听。这种思维方式可能在常人眼里非常诡异，但是小C确实更习惯于在脑子里面写好代码让自己的脑子计算。也正因为如此，小C的高中数学计算从来没错过。小C自己也不知道为什么自己和别人思维迥异，也不明白老师为什么经常说:"这个题计算量太大，我们换一种简单方法"。在他看来，只要不脑壳痛，想算到几位就算到几位并且速度很快。

- ...第三类呢，是每月固定量的套餐，有好几种套餐呢，第一种呢，是每月60分钟通话，用超了的话，不足一分钟按一分钟记，每分钟是1元;流量1G，如果超出1G，每M按照0.29元计费，如果累计8元，可以免费用至100M，如果累计50元，可以免费用至1G，如果免费额度又用尽了，那还按上面计算。这款套餐的基础费用很便宜，是58元...

>小C一个月要打m分钟电话，用nG流量，若使用第三类套餐第一种，请你帮忙写一个程序，输入两个浮点数类型数据，输出需要的钱数。（1G=1024M，计算M时到整数即可，不足的依然按1M计）

- 等等...小C想到:我平时用流量是很规律的呀，打电话也很有规律，那么我可以算出来我每个月用多少时长和流量呀！

>小C周一到周五大部分时间都在自习室和教室，是不需要流量的，只有晚上在宿舍的一个小时小C会用pM流量（每天）看看知乎刷刷B站。周六周日每天小C会用qM流量玩游戏。在周一，周五小C会和父母打电话x分钟y秒，这个月有z天，第一天是周u（u=1,2,3,4,5,6,7 7是周末）求总共通话时长f分d秒，和总共流量hG。


```
要求输入:
int z int u int x int y int p int q
输出:
int f int d double h
```

- 很快小C就挑好了套餐，正准备离开。抬头正对上那姑娘的目光。小C没有再看，转身出门。

- 回到学校大门旁边，小C只顾着拍照p图发朋友圈，丝毫没有注意到身后不知道什么时候多了一个人。

- “啊...你干嘛！”小C几乎要叫出来了，但是一看是刚才那个女生示意他小声一点。"怎么还是你？"女生没有回答，甩了甩她标志性的双马尾拉起他就走，反问道:"你刚才为什么那么快就选了一个最优的套餐呢？"，小C哭笑不得，心想，我选什么关你屁事。这话当然不能当着女生的面说，于是他说:"你放开我..怎么，有什么问题吗？"。"当然有"，那女生说着，脚下步子却是不停，"可不是所有人都能做到这样的。"小C母胎solo，哪有被女生拉着走过，一时间也没有挣脱。

- 走到一个没人的角落坐下，女生转过来对着他:"能说说你是怎么想的吗？"。小C经历过太多次这种问题了，有时候老师都惊叹于他惊人的计算能力，甚至怀疑他用了计算器。但是他不能说。如果说了，他会被孤立的，因为这实在是和其他人太不一样了。小C故作轻松:"害，哪有那么神，就是按感觉嘛..."，他本想着这种模糊的说辞能像往常一样糊弄过去，可是这一次，他错了。话还没说完，他就被像扭麻花一样硬生生扭了过来。没想到这个女生看起来柔弱，力气却是比他还大。

- 这是他第一次正视她的脸，前面的刘海稍微乱了一点，整体上看很匀称，除了那一双正瞪得铜铃似的大眼睛。"常清逸，2001年8月13号出生于H省F市，别以为我不知道你的底细"。她声音不大，但是却很有威慑力。"第100对孪生素数是多少？"...

>孪生素数对就是指两个相差为2的素数的整体。给定数k<200，求第k对孪生素数。

- ... "4019和4021"

- ..."第六个完全数是多少？"...

>完全数就是真因子的和等于它本身的一类数，输入k（k<=6）输出第k个完全数。

- ...小C说完就后悔了。但他当时脑子木木的，又被她力气和对自己的了解程度震惊了一下，完全没有反应过来自己被套路了。刚才这两个问题完全不是口算可以解决的，可是他却能一口报出答案，这绝非正常。

- 那女生带着胜利者的微笑:"现在可以说了吗？" 小C叹了口气，把自己的这个能力和盘托出。最后怕她觉得自己是编的，还小心翼翼的加了一句:"反正情况就是这个样子，信不信由你。"

- 让他没想到的是，女生点了点头:"我当然相信，要不然我为什么会盯上你。"顿了一顿，伸出了右手:"我叫谢山彤，很高兴认识你。"

- 第一章 完
